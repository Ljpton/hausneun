Informationen

Diese Seite dient zur Navigation durch das Haus 9 (auch "Papierfabrik" genannt) der Hochschule Harz.

Steuerung

Wenn die linke Maustaste gedrückt bleibt, kann das Modell verschoben werden. Mit der rechten Maustaste kann das Modell gedreht werden. Durch ein Scrollen oder das Gedrückthalten der mittleren Maustaste kann hinein- und hinausgezoomt werden.
Mit einem Finger kann das Modell verschoben und mit zwei Fingern gedreht werden. Mit einem auseinander- oder zusammenziehen der Finger kann hinein- und hinausgezoomt werden.

Ebenenwechsel

Ein Wechsel zwischen den Ebenen ist durch die Pfeile möglich. Die Ebenen gehen von 1-4. Das Erdgeschoss ist somit Ebene 1.

Neu zentrieren und 2D-/3D-Ansicht

Durch einen Klick auf das Zentrieren-Symbol kann das Modell neu zentriert werden oder von einer 3D- zu einer 2D-Ansicht gewechselt werden.

Suche

Die Suche kann durch einen Klick auf die Lupe gestartet werden.
Hier kann ein Raum eingegeben und mit Betätigen des Enter-Knopfes gesucht werden. Außerdem kann durch das Eingeben von zwei Räumen und dem Betätigen des Enter-Knopfes eine Routenplanung gestartet werden.

Infotexte und 360°-Ansicht

Mit dem Klick auf einen Raum fährt sich unten ein kleiner Infotext aus. Das Wählen des Raumes kann durch einen Klick außerhalb des Raumes wieder rückgängig gemacht werden.
Bei einigen Räumen kann eine 360°-Ansicht angewählt werden. Dies geht über das 360°-Symbol im Info-Bereich des Raumes.
